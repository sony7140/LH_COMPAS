## 1. 프로젝트 개요
### 문제정의
- 급격한 기후변화와 도시 인프라 노후화로 인해 기존의 사후 대응형 도시 관리 방식만으로는 교통안전 문제를 충분히 해결하기 어려움
- 특히 어린이,노인 등 교통약자는 스쿨존, 통학로 사고에 더 취약해 위험지역을 사전에 진단하고 예방형 대책을 마련할 필요가 있음

### 해결
- 교통사고 이력, 유동인구, 교통량·속도·혼잡도, 보행안전시설, 토지이용계획 데이터를 격자 단위로 통합하여 교통약자 위험지역을 정량적으로 분석
- Random Forest와 GWR를 비교한 뒤 GWR을 최종 모델로 선정했고 분석대상지에서 학습한 위험 패턴을 하남교산 토지이용계획도에 확장 적용해 설치 우선지역과 추천 시설물을 제안


## 2. 제공 데이터

### 공개 데이터
- 격자 데이터: `01_격자_(4개_시·구).geojson`, `02_격자_(하남교산).geojson`
- 인구 데이터: `03_성연령별_거주인구(격자).csv`
- 교통약자 및 안전시설 데이터: 어린이보호구역, 학교현황, 유치원현황, 어린이집현황, 횡단보도 위치정보, 버스정류장 위치정보, CCTV 현황, 과속방지턱 현황
- 공간계획 데이터: `22_토지이용계획도_(4개_신도시).geojson`, `23_토지이용계획도_(하남교산).geojson`
- 기타 참고 데이터: 하남교산 지구계획 변경자료, 교통데이터 코드표

### 비공개 데이터
- 성·연령별 유동인구, 시간대별 직장인구, 시간대별 방문인구, 주중주말 서비스인구
- 상세도로망 네트워크, 평균속도, 추정교통량, 혼잡빈도강도, 혼잡시간강도, 교통사고이력
- 비공개 데이터는 **LH COMPAS 환경에서만 API 호출 방식으로 활용**했으며, 외부 반출이 제한되어 저장소에는 포함하지 않음

### 실행 환경
- 본 프로젝트는 비공개 데이터 API 호출 때문에 **LH COMPAS 노트북 환경**에서 진행

## 3. 주요 인사이트
- **어린이 위험지역**은 교육시설 인접성, 통학로, 횡단보도, 생활도로와의 연관성이 크게 나타났습니다.
- **노인 위험지역**은 주거지·공공시설 접근부와 교차로 주변에서 상대적으로 위험이 높게 나타났습니다.
- 같은 교통약자라도 **어린이와 노인은 위험을 형성하는 공간적 조건이 다르게 나타났습니다.**
- 분석대상지에서 학습한 위험 패턴을 하남교산에 확장 적용하여, **개발 예정지의 사전 예방형 안전계획**에 활용 가능하도록 했습니다.

## 4. 주요 결과
- Child: RF(R² -0.2644) 대비 GWR(R² 0.6646)로 성능이 크게 개선됨
- Old: RF(R² 0.2399) 대비 GWR(R² 0.6146)로 더 높은 설명력을 확보함
- 최종적으로 성능과 공간 해석 가능성을 고려해 GWR을 최종 모델로 선정
- 하남교산 교통약자 설치 우선지역 시각화
 - 인터랙티브 지도에서 각 격자 hover 시 설치 우선점수와 추천 시설물, 설치 사유를 직관적으로 확인할 수 있도록 구성했습니다.
